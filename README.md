# Kali Helper

linux tool for Kali linux

A little bit about me: I want to say upfront that I realize this isn't exactly top-notch work, but it is my very first project—so I hope it proves at least a little bit helpful to you!

I'm 15 years old; I have about a month's worth of experience with C++, and slightly more with Python. Not long ago, I decided that I wanted to pursue a career in cybersecurity and embedded systems—and really, what kind of cybersecurity professional lacks an understanding of Linux, C++, and C?

I started by installing Kali Linux; personally, running apt update and upgrade took me two full days—non-stop—due, of course, to the lengthy download and installation times for all the packages. During that process, I got the urge to create something that could automate or simplify this whole procedure—and that's how it all began.

A couple of days ago, I switched over to Arch Linux running Hyprland; I've been thinking about building a similar utility based on this "Kali Helper" project—though, strictly speaking, that wouldn't really count as making any new progress,

since I need to gain knowledge in this subject. I hope this proves helpful to other beginners like myself, and that those with more experience can turn this jumbled mess into something coherent.
